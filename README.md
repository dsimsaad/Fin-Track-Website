# Smart Finance Tracker & Advisor (FINTRACK PLUS)

FINTRACK PLUS is a web-based personal finance management application built with **HTML, CSS, JavaScript, Bootstrap, and Firebase**. It helps users manage their income, expenses, savings, budgets, and investments through a simple and intuitive interface.

The goal of the project is to make personal finance management accessible to everyone, regardless of their financial or technical background. By organizing financial data in one place and providing helpful insights, FINTRACK PLUS encourages better financial habits and smarter decision-making.

---

## Features

* Secure user authentication with Firebase
* Income tracking
* Expense management
* Budget planning and monitoring
* Savings goals with progress tracking
* Investment tracking
* Financial reports and summaries
* Investment calculator
* Compound interest calculator
* AI-powered financial assistant ("Mark")
* Responsive design for desktop and mobile devices
* Automatic data persistence using Firebase and Local Storage

## How It Works

1. Create an account or log in.
2. Record your income and expenses.
3. Set budgets for different spending categories.
4. Track your savings goals and investments.
5. View financial summaries and charts.
6. Use the built-in calculators for financial planning.
7. Ask Mark for simple financial advice.
